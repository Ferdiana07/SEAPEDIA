## 🎯 Tujuan Pembelajaran

1. **Memahami fundamental** Full-Stack Development
2. **Membangun aplikasi lengkap** dari Frontend hingga Backend
3. **Belajar Laravel & React** dari dasar
4. **Desain database** yang proper
5. **API Design** yang RESTful
6. **Deployment** ke production

## 🔧 Tech Stack

| Layer | Teknologi |
|-------|-----------|
| Frontend | React + Tailwind CSS + Vite |
| Backend | Laravel (PHP) |
| Database | MySQL |
| Auth | Laravel Sanctum |
| State Management | Zustand |
| Routing | React Router DOM |
| HTTP Client | Axios |

## 📝 Cara Menggunakan Dokumentasi Ini

### Untuk Pemula:
1. Baca BAB 0-10 **secara berurutan**
2. Baca `_reference/` jika ada istilah yang belum dipahami
3. Kerjakan latihan di akhir setiap bab
4. Praktik langsung dengan coding
5. Bertanya jika ada yang tidak dipahami

### Referensi Cepat:
Jika kamu lupa apa fungsi suatu tools/framework, buka folder `_reference/` dan cari di sana.
